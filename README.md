# Minecraft Digital Circuit Saves | 我的世界数电存档合集

A collection of Minecraft digital circuit save files built with **vanilla items** (no pre-installed mods). Theoretically compatible with multiple versions, with **recommended usage on 1.16+** (untested on older versions, which may have unexpected bugs).

The first save file in the collection features a **superflat world** that includes both buildings and circuits (all-in-one content) and has **cheats disabled** by default. Supports Forge/Fabric environments: use WorldEdit/Create mod (optional) for blueprint copying, or Fabric-compatible blueprint tools.

这是一个基于**原版物品制作**的我的世界数电存档合集（无内置模组），理论上兼容多个游戏版本，**推荐在 1.16 及以上版本使用**（低版本未测试，可能出现异常 BUG）。

合集中的第一个存档为**超平坦世界**，内置建筑与电路（内容一应俱全），且默认**未开启作弊模式**。支持 Forge/Fabric 双环境：可自行安装创世神（WorldEdit）、机械动力（Create）模组实现蓝图拷贝，也可使用 Fabric 端兼容的蓝图工具导入。

## Key Features | 核心特性



* **Vanilla-Based Design**：All circuits are built with vanilla items, no mandatory mods required

* **原版物品构建**：所有数电装置均使用原版物品制作，无需强制安装模组

* **Version Recommendation**：Recommended for 1.16+ (older versions untested, potential bugs exist)

* **版本推荐**：推荐 1.16 及以上版本使用（低版本未测试，可能存在 BUG）

* **First Save Details**：Superflat world with integrated buildings & circuits, cheats disabled by default

* **首个存档详情**：超平坦世界，融合建筑与电路内容，默认未开启作弊

* **Dual-Loader Support**：Works on both Forge and Fabric, with optional mod integration

* **双加载器兼容**：支持 Forge 和 Fabric 环境，可选择性整合模组

* **Flexible Blueprint Tools**：


  * Forge: Optional WorldEdit/Create mod for blueprint copying

  * Fabric: Compatible with Fabric-specific blueprint tools

* **灵活蓝图工具**：


  * Forge 端：可自选安装创世神 / 机械动力模组实现蓝图拷贝

  * Fabric 端：支持 Fabric 专属蓝图工具导入

* **Modular Integration**：Independent circuit designs for easy migration to your own worlds

* **模块化整合**：数电装置均为独立设计，可轻松迁移到你的个人存档中

## How to Use | 使用说明



1. Download the desired save file from the `saves/` directory (check the file name for the "superflat" first save)

2. 从 `saves/` 目录下载所需存档文件（首个超平坦存档可通过文件名识别）

3. Place it in your Minecraft `saves` folder (e.g., `.minecraft/saves/`)

4. 将文件放入我的世界 `saves` 文件夹（例：`.minecraft/saves/`）

5. For vanilla/modded use: Launch the game (1.16+ recommended) directly (no mods needed); note that cheats are disabled in the first save

6. 原版 / 模组使用：直接启动游戏（推荐 1.16 + 版本）即可，无需安装模组；注意首个存档默认未开作弊

7. To enable cheats (if needed): Open the world to LAN in-game, then enable cheats in LAN settings

8. 如需开启作弊：在游戏内将世界对局域网开放，在局域网设置中启用作弊功能

9. For blueprint copying (Forge): Install WorldEdit + Create mod manually

10. 蓝图拷贝（Forge 端）：需手动安装创世神 + 机械动力模组

11. For blueprint copying (Fabric): Install Fabric-compatible blueprint mods

12. 蓝图拷贝（Fabric 端）：安装 Fabric 兼容的蓝图模组

## Notes | 注意事项



* **Version Warning**：Avoid using versions below 1.16 (untested, may cause functional bugs)

* **版本警告**：不建议使用 1.16 以下版本（未测试，可能导致功能异常）

* **Mod Optional**：WorldEdit/Create are only for blueprint operations, not required for circuit use

* **模组可选**：创世神 / 机械动力仅用于蓝图操作，数电本身使用无需这些模组

* **Vanilla Compatibility**：All circuits retain full functionality in pure vanilla environments

* **原版兼容性**：所有数电装置在纯原版环境中可保持完整功能

* **Cheat Status**：Cheats are disabled by default in the first superflat save; enable via LAN settings if required

* **作弊状态**：首个超平坦存档默认关闭作弊，如需开启可通过局域网设置操作

* **Bug Reporting**：Submit issues for version-specific bugs (especially below 1.16)

* **BUG 反馈**：若遇到版本相关 BUG（尤其 1.16 以下版本），可提交 issue 反馈
